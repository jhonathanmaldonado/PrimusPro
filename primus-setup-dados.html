🛠️ Primus — Setup de Dados

Execute na ordem. Feche depois que terminar.

Login Admin

Entre com a conta admin para executar as operações.

1. Limpar Dias de Teste

Apaga todos os documentos das coleções presencas_diarias, fechamento_diario e fechamento_funcionario.
Também limpa a coleção escala_semanal.
⚠️ Atenção: esta operação é irreversível. Todos os dados de teste serão apagados permanentemente.

2. Sincronizar Cadastro de Funcionários

Atualiza os 40 funcionários da lista oficial.
• Se já existe pelo nome → atualiza cargo e Pix
• Se não existe → cria novo com ID automático
• Funcionários que não estão na lista → marcados como inativos
